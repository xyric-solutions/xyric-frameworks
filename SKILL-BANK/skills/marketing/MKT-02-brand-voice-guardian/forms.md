# Brand Voice Guardian — Forms & Checklists

Ready-to-use intake, checklists, and review templates. The voice specifics (tagline,
attributes, banned/preferred words, key messages) come from your organization's brand
decisions; this file never hardcodes them.

---

## Voice Intake (the questionnaire for adapting this skill)

This is the questionnaire for adapting the skill to an organization (see
[`ADAPT.md`](ADAPT.md)). Ask as one numbered list and **WAIT** for answers; then record
the answers wherever your team keeps brand decisions. Ask only what's missing.

1. **Voice attributes** — the 3-5 adjectives that define how the org sounds (e.g.
   insightful, practical, confident)?
2. **Tagline** — the exact string including punctuation?
3. **Brand promise / positioning line** — one sentence the voice must live up to?
4. **Key messages** — the 5-9 points the voice should reinforce?
5. **Banned phrases** — words/phrases this org never uses (beyond the generic dead-word
   list)?
6. **Preferred phrases** — the on-brand replacements to swap in for transactional or
   buzzword language?
7. **Off-limits clichés / clichés to avoid** — anything else to never say?
8. **Relationship framing** — is this a service/partner org ("we're your team") or a
   product org? Shapes the partnership-voice table.
9. **Sides of the offer** — does the org do more than one core thing the voice must
   reflect?
10. **Account voices** — does anyone write under a personal/founder name distinct from
    the company voice? If so, capture each person's name, angle, and voice.

## Voice Checklist

Before publishing any content, verify. "Org X" = the organization's own values.

### Human Check
- [ ] Read it out loud — sounds like a real person talking?
- [ ] At least ONE specific detail (name, number, place, year)?
- [ ] Would you send this as a message to a smart friend?
- [ ] At least one casual, imperfect, or raw moment?
- [ ] First line stops the scroll?
- [ ] FREE of dead words and the org's banned phrases?

### Dead Words Check
- [ ] No "leverage" / "unlock" / "game-changer"
- [ ] No "in today's fast-paced world" / "let's dive in" / "here's the thing"
- [ ] No "synergy" / "holistic" / "paradigm" / "best-in-class" / "cutting-edge"
- [ ] No "I'm thrilled to announce" / "excited to share"
- [ ] No "at the end of the day" / "touch base" / "circle back"
- [ ] No phrase on the org's banned-phrases list
- [ ] If it sounds like a generic AI wrote it → kill it and rewrite

### Voice Alignment (one line per voice attribute adjective)
- [ ] **Attribute 1**: is the copy clearly this?
- [ ] **Attribute 2**: is the copy clearly this?
- [ ] **Attribute 3**: is the copy clearly this?
- [ ] **Relationship**: does this sound like a partner/team, not a transactional vendor?
- [ ] **Genuine**: would this feel "off" if it came from AI without human review?

### Promise / Tagline Check
- [ ] Sounds like it came from a team that owns its promise — not a generic agency?
- [ ] Tone matches what the brand calls for (proactive vs. measured)?
- [ ] Reflects every side of the offer the org describes?
- [ ] Tagline used verbatim, including punctuation?
- [ ] Could a competitor say the exact same thing? If yes — rewrite.

### Account Voice Check (only if writing under a personal/founder name)
- [ ] Uses "I" not "we" (personal, not company)
- [ ] Sounds distinctly like one named person — not interchangeable
- [ ] Matches that person's stated angle
- [ ] References the org naturally (not forced)
- [ ] Would NOT work on the company page (that's the test)

### Message Clarity
- [ ] Main point clear within the first sentence
- [ ] No jargon without context
- [ ] Benefit to the reader is obvious
- [ ] Call to action clear (if applicable)

### Brand Consistency
- [ ] Consistent with the org's other content
- [ ] Aligns with the visual brand (see the brand guidelines skill)
- [ ] Reinforces at least one of the org's key messages
- [ ] Speaks to a defined audience tier
- [ ] No contradictions with other messaging

## Content Transformation Checklist

When reviewing existing content for brand alignment:

```
BEFORE/AFTER REVIEW
- [ ] Replace transactional language with the org's relationship framing
- [ ] Replace buzzwords / banned phrases with specific outcomes
- [ ] Replace generic claims with methodology / evidence references
- [ ] Replace formal corporate tone with the org's direct tone
- [ ] Add HOW after any bold promise
- [ ] Remove unnecessary hedging
- [ ] Ensure the first sentence delivers the main point
```

## Success Criteria Checklist

| Criteria | Measure |
|----------|---------|
| Content sounds distinctly like the org | Anyone can tell its content from generic copy |
| Voice consistent across channels | Same personality in proposals, social, website |
| Relationship positioning clear | Audience describes the org as intended |
| Confidence backed by substance | No claims without methodology / evidence |
| Guidelines are actionable | Writers can apply without ambiguity |
