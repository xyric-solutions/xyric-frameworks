# Content Copy Engine — Forms & Checklists

Ready-to-use intake, checklists, and output templates. The voice, key messages,
content pillars, ICP tiers, palette, accounts, and founders are decided per
organization (see [`ADAPT.md`](ADAPT.md)); this file never hardcodes them.

---

## Content Context Intake (the questionnaire for adapting this skill)

Use this to gather everything the skill needs for an organization. Ask as one
numbered list and **WAIT** for answers; record the answers wherever your team keeps
brand and content decisions. Ask only what's missing.

1. **Accounts** — which accounts post content (the org plus any named founders)?
   For founders, capture a one-line voice note each (name, angle, voice).
2. **Brand promise** — one-line core value statement.
3. **Key messages** — your 5–9 recurring messages (one of them is your signature
   theme).
4. **Differentiators** — what makes you different / your value pillars.
5. **ICP tiers** — the audience segments you target (names + one-line each, NO
   pricing).
6. **Content pillars** — your 3–5 content pillars.
7. **Channels** — active platforms (LinkedIn / X / Instagram / etc.).
8. **Voice** — 3–5 voice adjectives and a words-to-avoid list.
9. **Tagline & descriptor** — exact tagline string and one-line "what we are".
10. **Palette** (for design briefs) — primary / accent / light / dark tokens.

## Human Check Validation Form

```markdown
## Human Check — [Post Title/Topic]

| # | Check | Result | Notes |
|---|-------|--------|-------|
| 1 | Sounds like a real person talking? | [ ] Pass [ ] Fail | |
| 2 | At least ONE specific detail? | [ ] Pass [ ] Fail | Detail: ___ |
| 3 | WhatsApp test — would you send this? | [ ] Pass [ ] Fail | |
| 4 | Imperfect/casual/raw moment? | [ ] Pass [ ] Fail | Where: ___ |
| 5 | First line stops the scroll? | [ ] Pass [ ] Fail | Hook: ___ |
| 6 | Dead words eliminated (baseline + your words-to-avoid list)? | [ ] Pass [ ] Fail | |

**Verdict**: [ ] Ready to post  [ ] Needs revision

**If Fail**: Rewrite flagged sections before output.
```

## Quality Gate Checklist (before final output)

| Gate | Check | Status |
|------|-------|--------|
| 1 | Human Check passed (all 6 items) | [ ] |
| 2 | Dead words eliminated (baseline + your words-to-avoid list) | [ ] |
| 3 | Design brief included (if visual post) | [ ] |
| 4 | ICP tier identified (from your ICP tiers) | [ ] |
| 5 | Value pillar reinforced (from positioning / content pillars) | [ ] |
| 6 | Correct account voice (org = "we", founder = "I" per the founder's voice note) | [ ] |
| 7 | Platform format rules followed (length, hashtags) | [ ] |
| 8 | Template adapted, not filled robotically | [ ] |
| 9 | Founder Personal Touch Check passed (if founder content) | [ ] |
