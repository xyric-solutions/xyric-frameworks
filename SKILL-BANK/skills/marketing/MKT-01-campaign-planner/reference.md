# Campaign Planner — Reference

Deep patterns, examples, and guidance, loaded on demand. Audience, messaging, channel,
and offering specifics are an organization's decisions (see [`ADAPT.md`](ADAPT.md)); the
cadence curves, funnel shapes, and theme escalation below are generic mechanics,
independent of any org.

---

## The organization-specific inputs

These are the values this skill operates on, gathered once per organization (see
[`ADAPT.md`](ADAPT.md) and the intake in [`forms.md`](forms.md)):

- **Positioning** — brand promise; key messages (one lead message per ICP tier / pain
  point); differentiators (proof points, methodology, credible outcome claims); ICP tiers
  (audience segments, each with a name and description).
- **Business** — business model (consultancy / SaaS / marketplace / …); service lines (each
  with a name and summary), mapped to pain points internally.
- **Content** — content pillars (3–5 recurring themes); active channels; founders (name,
  angle, voice) for per-founder content.
- **Brand voice** — tagline, tone attributes, and an "avoid" list.

Resolve every concrete value from these at use time. If a value isn't decided yet, treat it
as unanswered (ask via [`forms.md`](forms.md), or fall back to a labeled `PLACEHOLDER`).

## GTM Plan Generation: Deep Patterns

### Fresh Launch vs Continuation — full cadence guide

**Fresh Launch cadence curve** (brand going from 0 → live):

```
Week 1:  ████░░░░░░░░  ~5 posts   — "Launch the voice"
Week 2:  ████████░░░░  ~8–10      — "Build the rhythm"
Week 3:  ██████████░░  ~10–12     — "Full cadence"
Week 4+: ████████████  ~10–12+    — "Sustain & scale"
```

Why Week 1 is lighter in Fresh Launch:
- Profiles may still be getting set up (banners, bios, featured content)
- The team is testing brand voice for the first time
- Calibrate what works before going full volume
- Better to post 5 great pieces than 10 rushed ones

**Continuation cadence** (already posting — extending the plan):

```
Week N:   ██████████░░  ~10–12     — Full cadence from day 1
Week N+1: ██████████░░  ~10–12     — Maintain momentum
Week N+2: ████████████  ~12–15     — Scale if team bandwidth allows
Week N+3: ████████████  ~12–15     — Optimize based on data
```

Why there's NO ramp in Continuation:
- Brand voice is already established
- The audience expects consistent posting
- Dropping cadence signals inconsistency
- Profiles, templates, and workflows are ready

> The post counts above are default heuristics. If the org or user sets a different
> volume, use theirs.

### Continuation Context Checklist

Before writing any continuation plan, gather:

```
1. PREVIOUS PLANS
   - [ ] Read each existing GTM plan (company + one per founder)
   - [ ] Note themes, formats, KPIs from each
   - [ ] Check "What Comes Next (Week 3+)" sections — these are PROMISES to honor

2. THEMES USED
   - [ ] List all weekly themes from previous plans
   - [ ] Generate NEW themes that build on previous ones (don't repeat)
   - [ ] Follow theme escalation: Foundation → Credibility → Authority → Conversion

3. FORMAT TRACKING (CUMULATIVE)
   - [ ] Count total uses of each format across ALL plans
   - [ ] Identify underused formats to rotate in
   - [ ] Introduce 1–2 NEW formats not yet used (carousels, newsletters, etc.)

4. PERFORMANCE DATA
   - [ ] If KPIs have "Actual" columns filled → analyze what worked
   - [ ] Double down on high-performing formats
   - [ ] Adjust or retire underperforming content types
   - [ ] Set new KPI targets relative to previous actuals (not from zero)

5. SCALING OPPORTUNITIES
   - [ ] Introduce carousel posts (Week 3+)
   - [ ] Start a newsletter (Week 4+)
   - [ ] Scale audio/podcast frequency (weekly)
   - [ ] Add guest collaborations / cross-promotions
   - [ ] Introduce lead magnets / downloadables
```

### Theme escalation pattern

Themes progress through a natural arc:

| Phase | Weeks | Theme pattern | Example theme shape |
|-------|-------|---------------|---------------------|
| **Foundation** | 1–2 | "Who we are" + "What we believe" | "[The core gap we see]", "[Hype vs reality]" |
| **Credibility** | 3–4 | "What we've done" + "How we think" | "[Patterns we've seen]", "[Our methodology]" |
| **Authority** | 5–6 | "Industry opinions" + "Predictions" | "[The future of X]", "[Why most projects fail]" |
| **Conversion** | 7–8 | "What this means for you" + "Next steps" | "[Are you ready?]", "[The cost of waiting]" |

For continuation plans starting at Week 3+, skip Foundation entirely — jump to
Credibility/Authority. Fill the example shapes from the org's key messages.

## Campaign Type Deep-Dives

### Type 1: Thought Leadership

**Purpose**: Position the org as the authority in its domain.

```
Duration: Ongoing (quarterly themes)
Primary Channel: top of the org's active channels (organic)
Secondary Channel: owned content / SEO (long-form)

Content Flow:
1. Insight Post     -> Establish expertise
2. Problem Post     -> Validate the pain point
3. Methodology Post -> Show the approach (the org's differentiators)
4. Story Post       -> Success pattern (anonymized or with permission)
5. Engagement       -> Respond, discuss, build relationships

Cadence:
- Primary network: 3–4 posts/week
- Long-form content: 1–2 pieces/month
```

**Quarterly theme shape** (fill the pain-point focus from the org's positioning):

| Quarter | Pain-point focus | Content theme shape |
|---------|------------------|---------------------|
| Q1 | `[Foundation-type pain point]` | "New year, clean slate" / planning |
| Q2 | `[Speed-type pain point]` | "Faster execution for the quarter" |
| Q3 | `[Scale-type pain point]` | "Growing pains? Optimize before you hire" |
| Q4 | `[Accuracy/trust pain point]` | "Year-end: is your [X] ready for review?" |

### Type 2: Problem-Focused Lead Generation

**Purpose**: Attract potential customers by addressing their specific pain points.

```
Duration: 4–6 weeks per campaign
Primary Channel: owned content / SEO + primary network

Funnel:
1. ATTRACT: Educational content about the problem
   - Blog post: "Why [problem] costs you more than you think"
   - Primary network: problem-focused posts, engagement

2. EDUCATE: Deeper methodology content
   - Guide/whitepaper: "How to solve [problem]: a practical approach"
   - Primary network: behind-the-scenes, methodology insights

3. CONNECT: Soft call-to-action
   - Discovery-call invitation (not a sales pitch)
   - "Let's discuss your specific situation"

Key Assets:
- Long-form educational content (ungated initially)
- A post series (5–7 posts)
- A simple discovery-call booking page
```

### Type 3: Customer-Story Amplification

**Purpose**: Share success patterns without being salesy.

**Approach**:
- Focus on the PROBLEM and TRANSFORMATION, not the org
- Use anonymized patterns if formal case studies aren't ready
- Let the results speak

**Format options**:
1. "Pattern we've seen" post (anonymized)
2. Before/after story (with permission)
3. Lessons learned (what worked, what didn't)
4. Methodology in action (how we approached it)

**Story framework**:
```markdown
## Customer Story: [Anonymized Name or Pattern]

### The Situation
[What was happening before — focus on their world]

### The Challenge
[Specific problem they faced — quantify if possible]

### The Approach
[How we worked together — partnership framing]

### The Outcome
[Results — specific numbers when available]

### The Lesson
[What this teaches about solving similar problems]

---
Note: mark any metric awaiting confirmation as PLACEHOLDER.
[CASE STUDY PENDING: specific metrics to be confirmed]
```

## Channel Strategy Details

Resolve the actual channel names and priority from the org's active channels. The sections
below describe generic channel archetypes.

### Primary: professional network (organic)

**Purpose**: Thought leadership and relationship building.

| Element | Guideline |
|---------|-----------|
| Tone | From the org's brand voice attributes |
| Personal vs company | Personal profiles often outperform the company page |
| Content mix | 60% Core, 30% Personality, 10% Wildcard |
| Frequency | 5–10 posts/week (company + founders combined) |
| Hashtags | 3–5 relevant, not trending-chasing |

**Content types**: problem-focused insights; methodology explanations; industry
observations; quick tips/frameworks; anonymized customer patterns.

**What we don't do**: service announcements; self-congratulatory posts; generic hype;
engagement bait.

### Secondary: owned content / SEO

| Element | Guideline |
|---------|-----------|
| Format | Blog posts, guides, methodology docs |
| Length | 1,500–3,000 words |
| Frequency | 1–2 pieces/month |
| Focus | Problem-first, solution-second |
| SEO | Target pain-point keywords, not service keywords |

### Secondary: real-time micro-posts

| Element | Guideline |
|---------|-----------|
| Tone | Punchy, direct (per the org's brand voice) |
| Energy | Hot takes, contrarian views, immediate reactions |
| Content mix | 60% Core, 30% Personality, 10% Wildcard |
| Frequency | Daily presence (1–4 posts/day) |
| Hashtags | Minimal (0–2 max) |

Micro-posts = faster, sharper, more immediate. Same pillars, different energy.
Cross-adapt insights to the format — not copy/paste.

### De-emphasized channels

**Email**: Use for existing relationships, not acquisition (unless the org's plan says
otherwise).
**Paid advertising**: Default to organic authority-building; add paid only if the org's
channel mix includes it.

## Messaging Principles

### Transformation examples

**Generic → on-brand**:
- Before: "We offer [service] development services"
- After: "Your team is spending 3 days on a report that should take 3 minutes. Here's why."

**Vendor → partner**:
- Before: "Choose us for your [domain] needs"
- After: "We become your embedded [domain] team — not another vendor to manage"

**Hype → substance**:
- Before: "AI is transforming business"
- After: "Document processing dropped from 2 hours to 4 minutes. Here's how."

> Replace the bracketed nouns with the org's business and positioning values; never invent
> a metric — pull proof points from the org's differentiators.
