# LinkedIn Content — Forms & Checklists

Ready-to-use intake, checklists, and templates. The brand voice, content pillars, audience
tiers, and key messages are an organization's decisions (see [`ADAPT.md`](ADAPT.md)); this
file never hardcodes them.

---

## LinkedIn Content Intake (the questionnaire for adapting this skill)

This is the questionnaire for adapting this skill to an organization. Ask the items as one
numbered list and **WAIT** for answers; then record them wherever your team keeps brand and
content decisions. Ask only what's not yet decided.

1. **Tagline & descriptor** — exact tagline string (incl. punctuation) and one-line
   descriptor?
2. **Voice attributes** — 3–5 adjectives that define how the brand sounds?
3. **Off-limits** — words, clichés, or styles to avoid (beyond the universal Dead Words)?
4. **Content pillars** — 3–5 recurring themes, each with its core insight?
5. **Channels** — which channels are active, and is LinkedIn company + founder both in play?
6. **Founders** — for any founder posts: name, audience angle, and personal voice per founder?
7. **Audience tiers (ICP)** — who are you speaking to (each tier: name + description + message angle)?
8. **Key messages** — 5–9 messages or differentiators to weave into posts?
9. **Brand promise** — one core value statement?

> If a marketing voice/brand skill (e.g. brand guidelines) already captured the brand voice
> or positioning, reuse those values instead of re-asking.

## Quality Checklist

Before posting, verify:

### Human Check (mandatory)
- [ ] Read it out loud — sounds like a real person talking?
- [ ] At least ONE specific detail (name, number, place, year)?
- [ ] Would you send this as a message to a smart friend?
- [ ] At least one imperfect, casual, or raw moment?
- [ ] First line stops the scroll?
- [ ] FREE of Dead Words? (see kill list below)

### Dead Words Check
- [ ] No "Leverage" (use "Use" or remove)
- [ ] No "Unlock" (use "Find" / "Build" / "Get")
- [ ] No "Game-changer" (say what actually changed)
- [ ] No "In today's fast-paced world" (delete, start with the point)
- [ ] No "Let's dive in" / "Here's the thing" (delete, just start)
- [ ] No "Synergy" / "Holistic" / "Paradigm" (use normal words)
- [ ] No "I'm thrilled to announce" / "Excited to share" (just share it)
- [ ] Nothing from the organization's own off-limits list

### Voice & Positioning
- [ ] Uses "we" not "I" (company voice — unless a founder post for a named founder)
- [ ] Sounds confident, not desperate
- [ ] Distinctly this brand — would not fit on any competitor's page
- [ ] Demonstrates expertise without claiming it
- [ ] Reinforces at least one of the organization's key messages / differentiators

### Content Strategy Check
- [ ] Maps to one of the organization's audience tiers — who is this speaking to?
- [ ] Fits the content mix (Core / Personality / Wildcard)
- [ ] Different format from the previous post (format diversity rule)
- [ ] One clear insight or takeaway
- [ ] Passes the "so what?" test
- [ ] Would a busy executive read to the end?

### Anti-Pattern Check
- [ ] No dramatic ellipses or "let that sink in"
- [ ] No engagement bait endings ("Drop a heart if you agree!")
- [ ] No buzzwords without substance
- [ ] No service announcements or vendor language
- [ ] No same format twice in a row

### Format
- [ ] Mobile-readable (short paragraphs, 1–2 sentences max)
- [ ] 3–4 hashtags maximum, at END of post only
- [ ] 0–2 emojis max, strategic only
- [ ] Clear hook in first line (scroll-stopping)

## Hashtag Strategy

Pick 3–4 hashtags drawn from the brand's domain and the post topic. Choose them from the
content pillar and the audience tier the post serves.

### Rules (universal)
- Maximum 3–4 hashtags per post.
- Always at the END of the post; never inline.
- No vanity branded hashtags (e.g. `#[CompanyName]`) — they rarely drive reach.
- Keep a small reusable set per pillar plus 1–2 topic-specific tags per post.

### Hashtag set worksheet (fill from your organization's pillars)
```

Topic-specific (per pillar):
  Pillar A → #___ #___
  Pillar B → #___ #___
  Pillar C → #___ #___
```

## Post Templates (universal structures)

> For brand-specific copy templates and tone, pair with the organization's voice/copy
> skill. These are LinkedIn-specific structural skeletons.

### Insight Post Template
```
[Hook: One-line observation that stops the scroll]

[Context: Why this matters — 1-2 sentences]

[Substance: The insight, pattern, or methodology — 3-5 short paragraphs]

[Implication: What this means for the reader]

#Hashtag1 #Hashtag2 #Hashtag3
```

### Methodology Post Template
```
[Hook: Common misconception or wrong approach]

[The problem with the common approach]

[Your framework / methodology — explained simply]

[What changes when you apply this]

#Hashtag1 #Hashtag2 #Hashtag3
```

### Pattern Story Template
```
[Hook: "A pattern we see in [type of business]:"]

[Setup: What was happening — specific details without identifying info]

[What most people do (and why it fails)]

[What actually works — your approach]

[Result: Specific outcome with numbers if possible]

[The lesson: Broader takeaway for the reader]

#Hashtag1 #Hashtag2
```

### Industry Commentary Template
```
[Hook: React to a trend, news item, or common belief]

[Your take: Why most people are thinking about this wrong]

[Evidence: What you've seen in practice]

[Conclusion: Sharp, memorable takeaway]

#Hashtag1 #Hashtag2 #Hashtag3
```

### Design Brief Template (for every visual post)
```
DESIGN BRIEF: [Visual type] | [Key text/quote for visual] | [Mood/style]
Brand colours / fonts / logo: use the organization's brand palette, typography, and logo
Logo: brand logo bottom-right (subtle, not dominant)
```

| Post Type | Brief Style |
|-----------|-------------|
| Contrarian Take | Quote card — key line as hero text, dark bg, bold light text |
| Myth Buster | Numbered list graphic — one myth per block, brand colours |
| Story Hook | No graphic — text-only performs better for stories |
| Team Series | Photo + welcome text overlay, brand frame, warm tone |
| Carousel | X slides — hook on slide 1, one point per slide, logo on last |
| Culture / BTS | Casual photo or phone shot — authenticity > polish |
| Engagement Q | Clean text card with the question — minimal, inviting |

## Success Metrics (universal targets)

### Engagement Quality (Over Quantity)

| Metric | Target | Notes |
|--------|--------|-------|
| Engagement rate | 3–5%+ | Quality of comments matters more than count |
| Profile views | Trending up | Indicates content attracting the right audience |
| Connection requests | From target audience | Decision-makers, not vendors |
| Inbound inquiries | 1+/month | The real goal |

### Content Performance Signals
- **High-quality signal**: comments that engage with the idea.
- **Low-quality signal**: generic "Great post!" comments.
- **Best signal**: DMs asking to learn more.
