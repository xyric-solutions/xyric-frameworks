# Cold Outreach — Forms & Checklists

Ready-to-use intake, brief, pre-send checklist, follow-up cadence, and tracker. The
org-specific values (name, positioning, voice, service lines) are decided per organization —
see [`ADAPT.md`](ADAPT.md); this file never hardcodes them.

---

## Cold Outreach Intake (the questionnaire for adapting this skill)

Ask as one numbered list and **WAIT** for answers; then record them wherever your team keeps
outreach decisions (an outreach brief, a positioning doc, or your fork of these files). Ask
only what's missing.

1. **Org name & one-liner** — exact name and a one-sentence elevator description?
2. **Descriptor** — the one-line "what we are" that sits under the tagline?
3. **Tagline** — exact string, including punctuation?
4. **Voice attributes** — 3-5 adjectives for how the org sounds?
5. **Off-limits words/phrases** — anything to never say?
6. **Differentiators** — what makes the org different from competitors, stated as concrete
   proof points (speed, ownership, who-you-talk-to, specialization)?
7. **Key messages** — the 5-9 core messages to draw from?
8. **ICP tiers** — the prospect segments you target, with a one-line description each, and
   how messaging shifts per tier?
9. **Business model** — one-line model; does the org have more than one "side" (e.g.
   optimize-existing vs. build-from-zero)?
10. **Service lines** — each offering with a one-line summary?
11. **Sign-off** — who signs cold emails (first name + role)?

## Pre-Send Checklist (Mandatory)

Run through ALL checks before sending. If any fail — fix before sending.

### Hook Check
- [ ] Subject line is value-first or curiosity-driven (NOT "Partnership Opportunity")
- [ ] First sentence mentions THEM, not us
- [ ] First 2 lines make the reader stop — there's a hook
- [ ] Subject line is under 50 characters (fits mobile preview)

### Human Check
- [ ] Read it out loud — does it sound like a real person talking?
- [ ] At least ONE specific detail (name, number, place, company)
- [ ] Would you send this as a message to a smart colleague?
- [ ] Is it FREE of Dead Words? (see kill list)
- [ ] Does it match the org's voice attributes, not a generic agency tone?

### Dead Words Check
- [ ] No "I hope this finds you well"
- [ ] No "I'm reaching out because"
- [ ] No "Partnership opportunity" (especially in subject)
- [ ] No "Leverage" / "Unlock" / "Comprehensive solutions"
- [ ] No "End-to-end" / "Data-driven insights"
- [ ] No "Best regards" / "Kind regards"
- [ ] No "Would you be available"
- [ ] No "I'd love to connect"
- [ ] No "Excited to share" / "Thrilled to announce"
- [ ] Nothing from the org's words-to-avoid list
- [ ] If it sounds like an LLM wrote it → kill it and rewrite

### Structure Check
- [ ] Under 200 words total (150 is better)
- [ ] Scannable in 30 seconds (short paragraphs, bullets, bold headers)
- [ ] No two back-to-back bullet lists without a header break
- [ ] Results section is at the END (the punchline)
- [ ] CTA reframes the call as VALUE, not an ask
- [ ] Sign-off is just first name + role + org name (from the org's decided sign-off)

### Results Check
- [ ] Every metric is computed from THIS prospect's situation
- [ ] No fixed boilerplate percentage shipped as a guarantee
- [ ] Unverified figures are bracketed (`[X]%`, `[timeframe]`)
- [ ] No invented client name, case study, or product name
- [ ] Directional language used where a hard number doesn't exist

### Targeting Check
- [ ] ICP tier identified (from the org's ICP tiers)
- [ ] At least one differentiator reinforced (from the org's differentiators)
- [ ] Company name mentioned 2+ times
- [ ] At least one personalized detail that shows research
- [ ] "Who we are in N seconds" block is specific to the service line being offered

### Distinctiveness Check
- [ ] Does this email feel like it came from this org and no other?
- [ ] Is there a concrete proof point for the org's lead differentiator?
- [ ] Would a competitor be able to send this exact email? (If yes — rewrite)
- [ ] If the model has sides, does it reflect the right one for this prospect?

## Follow-Up Cadence

After the first email, follow this schedule:

| Touch | Timing | Format | Content |
|-------|--------|--------|---------|
| **Email 1** | Day 0 | Cold email | Full outreach (Template 1-4) |
| **Follow-Up 1** | Day 3-4 | Reply to original | Short bump — "figured this got buried" |
| **Follow-Up 2** | Day 7-8 | New angle | Different value prop or comparable scenario |
| **Follow-Up 3** | Day 14 | Break-up email | "If timing's off, no stress. Reply 'later' and I'll check back in [X]" |
| **Social DM** | Day 1-2 | Connection request | Short note referencing the email |

### Follow-Up Rules
- **Never** send more than 4 touches total
- **Never** repeat the same pitch — each follow-up adds new value
- **Always** reply to the original thread (keeps context)
- **Break-up email** gives them permission to say "not now" — reduces pressure
- **Social connection** within 48 hours of first email — light touch, not a sales pitch

## Follow-Up Templates

Square-bracket values are per-prospect; `[First Name]` / role / org sign-off come from the
org's decided sign-off.

### Follow-Up 1: The Bump (Day 3-4)

```
Subject: Re: [Original Subject]

Hey [First Name],

Bumping this up — figured it might have gotten buried.

Quick version: We help [industry] companies save [X] hours/week on
[specific process]. Done it for [number] companies this year.

If timing's off, no stress. Just let me know.

[First Name]
```

### Follow-Up 2: New Angle (Day 7-8)

```
Subject: Re: [Original Subject]

Hey [First Name],

Different angle — we just finished a project for a [similar company type]
where we [specific deliverable]. Took [X] weeks. They were seeing results
by week [X].

Thought it might be relevant for [Company] given [specific reason].

Happy to send a 1-page breakdown if useful?

[First Name]
```

### Follow-Up 3: The Break-Up (Day 14)

```
Subject: Re: [Original Subject]

Hey [First Name],

Last note from me — I get it, inboxes are brutal.

If [specific problem] is on your radar for later this year, just reply
"later" and I'll circle back at a better time.

Either way — wishing [Company] a great [quarter/year].

[First Name]
```
