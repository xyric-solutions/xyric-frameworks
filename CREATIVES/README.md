# Creatives

An index-first library for inspiration, references, assets, examples, and creative quality bars.

The folder is intentionally lightweight. Keep the searchable index in git, and store heavy files in Google Drive or another asset store.

## Recommended Model

```text
Google Drive or asset store
  screenshots/
  brand-assets/
  decks/
  fonts/
  templates/

CREATIVES/
  creatives-index.yaml
  creatives-index.template.yaml
  README.md
  CONTRIBUTING.md
```

## How To Use

1. Copy `creatives-index.template.yaml` to `creatives-index.yaml` in your repo or workspace.
2. Add one entry per reference.
3. Store heavy files outside git.
4. Keep the `why` field specific enough that a future human or AI agent knows why the reference matters.

## AI Guidance

- Use references for inspiration, comparison, and quality calibration.
- Do not copy an asset, layout, brand system, or creative exactly unless the license and owner allow it.
- Preserve attribution and usage notes.
- Prefer a small number of high-signal references over a large dump of unlabeled links.

No public Drive folder is linked yet. Add one only when the asset store is intentionally public.
